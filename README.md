# Personal Brand Portfolio

A modern, minimalist portfolio website built with vanilla JavaScript and GSAP animations.

## Features

- Responsive design
- Smooth animations
- Category filtering
- Dark mode support
- Mobile-first approach

## Structure

```
├── css/
│   ├── base/
│   │   ├── variables.css
│   │   └── reset.css
│   ├── components/
│   │   ├── loading.css
│   │   ├── navbar.css
│   │   └── minimal.css
│   └── layout/
│       └── sections.css
├── js/
│   ├── core/
│   │   ├── LayoutManager.js
│   │   ├── StateManager.js
│   │   └── BaseAnimationController.js
│   ├── animations.js
│   └── App.js
└── index.html
```

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start customizing!
